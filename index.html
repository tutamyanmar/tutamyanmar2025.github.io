GitHub Pages ကိုအသုံးပြုပြီး တည်ဆောက်ထားသော ဝက်ဘ်ဆိုက်

  • ပင်မစာမျက်နှာ
  • အကြောင်းအရာ
  • ဝန်ဆောင်မှုများ
  • ဆက်သွယ်ရန်

GitHub Pages အကြောင်း

GitHub Pages သည် GitHub မှပေးသော အခမဲ့ဝန်ဆောင်မှုတစ်ခုဖြစ်ပြီး ကိုယ်ပိုင် website များကို တည်ဆောက်နိုင်ရန် အထောက်အကူပြုပါသည်။

အသေးစိတ်ဖတ်ရန်

အကျိုးကျေးဇူးများ

GitHub Pages ကိုအသုံးပြုခြင်း၏ အကျိုးကျေးဇူးများစွာရှိပါသည်။ အခမဲ့ဖြစ်ခြင်း၊ လွယ်ကူစွာတည်ဆောက်နိုင်ခြင်း၊ SSL Certificate အပါအဝင်ဖြစ်ပါသည်။

ဆက်လက်လေ့လာရန်

စတင်အသုံးပြုနည်း

GitHub Pages ကိုစတင်အသုံးပြုရန် GitHub account တစ်ခုဖန်တီးပြီး repository အသစ်တစ်ခုကိုဖန်တီးရန်လိုအပ်ပါသည်။
